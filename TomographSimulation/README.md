# Tomograph Simulation


Tomograph Simulation in python and jupyter notebook.
Uses:
* Bresenham Algorithm
* Radon Transform
* Inverse Radon Transform

## Usage

GUI created as Jupyter Notebook
Uses python modules:

* numpy
* matplotlib
* Pillow
* IPython
